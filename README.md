&#x20; **E-Commerce Sales data Analysis Dashboard**



This project analyzes E-Commerce sales data using Python and Jupyter Notebook. The dashboard provides insights into sales performance, profit trends, regional performance, product categories, and customer purchasing behavior.



**Objectives**

* \- Analyze sales and profit performance.
* \- Identify top-performing products and categories.
* \- Compare sales across different regions.
* \- Track monthly sales and profit trends.
* \- Visualize key business metrics using charts.



&#x20;**Tools \& Technologies**

\- Python

\- Jupyter Notebook

\- Pandas

\- NumPy

\- Matplotlib

\- Seaborn



**Dataset Information**



**The dataset contains the following columns:**



* Order Date
* &#x20;Product Name
* &#x20;Category
* &#x20;Region
* &#x20;Quantity
* &#x20;Sales
* &#x20;Profit

**Category-wise Sales**
